Généralités sur les interfaces

Les interfaces d'administation reprennent toute le même schéma. La première page permet de sélectionner l'élément sur lequel agir. La deuxième (et/ou la troisième) permet de modifier les données de l'élément. Certaines interfaces en première page montrent plus d'informations que d'autres.
