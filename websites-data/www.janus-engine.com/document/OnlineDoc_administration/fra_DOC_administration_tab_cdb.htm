Commande de base
site_context C'est l'une des commandes les plus importante. Elle permet de sélectionner le site sur lequel on va appliquer les commandes suivantes.

Exemple:

> site_context "MultiWeb_Engine";
Ajout Ajoute un élément. Est soumis a la condition universelle du nom de l'élément. Chaque nom d'élément doit être unique dans sa catégorie.
Modification Modifie un élément. La sélection de l'élément est soumise à la condition qu'il existe.
Suppression Supprime un élément. La sélection de l'élément est soumise à la condition qu'il existe.
Affiche Affiche le détail d'un élément. La sélection de l'élément est soumise à la condition qu'il existe.
Commande spécialisées
insert_content Manipule le document en remplaçant le contenu de celui par le contenu d'un fichier sélectionné.
share_document Permet le partage le document avec un autre site. Seul le propriétaire d'un site peut le faire pour ses documents et/ou le propriétaire du serveur.
user Initie une action sur un utilisateur (ex: join_group).
Link Commande dédiée aux article qui permet d'associer un document à un article.
