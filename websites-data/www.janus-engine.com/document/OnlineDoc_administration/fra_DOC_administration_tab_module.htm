« module » : Représente un module.

Les arguments associés à l'élément « module »
nom Nom qui servira pour le référencement.
titre Champ libre.
file Champ de type ficher. Défini le fichier qui apportera le contenu de ce module.
desc Description.
group_who_can_see Nomme le group qui peut voir le module.
group_who_can_use Nomme le group qui peut utiliser le module.
deco OUI / NON
deco_nbr Numéro de la décoration a utiliser
position Position dans la file de traitement des modules.
filtre Champ libre. Permet de filter le résultat de la commande "show" sur le nom interne des éléments cible.

Exemple:

> site_context "MultiWeb_Engine"; > add module name "authentification" title "Authentification" file "modules/module_authentification.php" desc "Formulaire pour se logger" group_who_can_see "Anonyme" group_who_can_use "Anonyme" deco "ON" deco_nbr "1" position "1";
