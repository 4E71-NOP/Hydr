« module » : Designate a module.

Associated arguments of « module »
name Name used as internal reference.
title / titre Text field.
file File field. Defines the file to be used for module content.
desc Description.
group_who_can_see Group name who can see the module.
group_who_can_use Group name who can use the module.
deco OUI / NON (YES / NO)
deco_nbr Decoration number to use.
position Module position in the render queue.
filter Text field. Enable filtering on the internal name of the target elements for the "show" command line.

Example:

> site_context "MultiWeb_Engine"; > add module name "authentification" title "Authentification" file "modules/module_authentification.php" desc "Formulaire pour se logger" group_who_can_see "Anonyme" group_who_can_use "Anonyme" deco "ON" deco_nbr "1" position "1";
 /*JanusEngine-Content-End*/
