« menu » : Designate menu elements.

Associated arguments of « menu »
name / nom Name used as internal reference.
parent Defines parent menu.
position Defines menu postion in the menu branch.
type 4 aviable types : MENU_ADMIN_RACINE MENU_ADMIN ARTICLE_RACINE ARTICLE.
lang Defines menu language. It will appears in the menu only if matching language have been selected.
state / etat ONLINE / OFFLINE
deadline / deadline Defines the associated deadline.
title / titre Text field. Appears in the menu.
desc Description.
group / group Defines which group is authorized to see the menu (and to use it).
doc_premier / first_doc Define the article to use incase of none has been selected during navigation.
article Use 0 to make it a directory; else, define the article reference.
filter Text field. Enable filtering on the internal name of the target elements for the "show" command line.

Exemple:

> site_context "MultiWeb_Engine";
> add menu name "fr_sommaire_admin" parent "0" position "01" type "MENU_ADMIN_RACINE" lang "FRA" state "ONLINE" deadline "initial_online" title "Sommaire" desc "Root menu" group "Anonyme" article "0" first_doc "YES";
