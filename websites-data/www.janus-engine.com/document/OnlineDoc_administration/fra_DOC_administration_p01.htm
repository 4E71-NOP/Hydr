L'administration des sites peut se faire de deux manières différentes. L'une repose sur l'interface HTML déjà présente; l'autre est orientée script et ligne de commande. L'une ou l'autre des méthodes offre différents avantages. L'interface HTML est bonne pour le tri des données et permet des recherches fines. Les scripts permettent de faire des traitements en batterie.

La méthode utilisant l'interface

L'interface s'avere utile lorsque l'on cherche des éléments spécifiques. Lorsque l'élément a été trouvé il peut être modifié a volonté. Il y a plusieurs menus (dans la partie admin) dédiée a la plupart des type d'éléments requis par le moteur. L'interface ne gère qu'un seul élément a la fois. Il est donc plus orienté a la modification ponctuelle. Pour régler un probleme rapidement sera adéquat de passer par l'interface.

Le méthode utilisant les lignes de commande

Les lignes de commande sont la véritable essence du moteur. Le menu dédié dans la partie administration est une interface. Un simple cadre permet d'entrer les commandes. On peut copier / coller des scripts entier et cliquer sur 'soumettre'. Bien évidement il y a une syntaxe et des règles. La plupart de ces règles sont simples. Chaque commande doit être terminée par « ; ». Chaque commande est éxécutée dans un contexte de site. Faites donc attention a être dans le bon contexte.
