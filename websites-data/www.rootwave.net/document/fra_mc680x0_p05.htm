/*Hydr-Content-Begin*/ Les instructions du mc680x0 de O à R

OROu exclusif. Réalisation d'un OU logique entre l'opérande source et l'opérande destination et rangement du résultat dans l'emplacement destination. OR ,Dn ou OR Dn,.
Variantes :
Réalisation d'un OU logique entre la donnée immédiate et l'opérande destination. ORI #,Dn.
Réalisation d'un OU logique entre la donnée immédiate et le SR. Cette instruction nécessite obligatoirement le mode superviseur du processeur ORI #,SR.

 
PEASauvegarde de l'adresse effective sur la pile (SP ou A7). Empilement de l'adresse effective. L'opérande concerne toujours les 32 bits du registre : PEA .
 

 
RESETRemise à zéro de tous les circuits extérieurs. Aucun autre état du processeur n'est affecté. L'exécution se poursuit a l'instruction suivante.
 
ROLRotation gauche. Rotation vers la gauche de l'opérande. Le bit de poids fort est dupliqué dans le bit C du CCR. Si l'opérande est un registre on spécifie le nombre de tours, soit par un registre (ROL Dn,Dd permute Dd le nombre de fois indiqué par Dn), soit par une donnée immédiate (ROL #,Dd avec une valeur de 0 a 7 pour la donnée). Si l'opérande est une case mémoire, un seul tour s'effectue et l'opérande ne peut être qu'un mot. : ROL .
 
RORRotation droite. Rotation vers la droite de l'opérande. Le bit de poids faible est dupliqué dans le bit C du CCR. Si l'opérande est un registre on spécifie le nombre de tours, soit par un registre (ROR Dn,Dd permute Dd le nombre de fois indiqué par Dn), soit par une donnée immédiate (ROR #,Dd avec une valeur de 0 a 7 pour la donnée). Si l'opérande est une case mémoire, un seul tour s'effectue et l'opérande ne peut être qu'un mot. : ROR .
 
ROXLRotation gauche avec le bit d'extension. Rotation étendue vers la gauche de l'opérande. Le bit de poids fort est dupliqué dans les bits C et X du CCR qui, a son tour, est dupliqué dans le bit de poids faible de l'opérande traité. Si l'opérande est un registre on spécifie le nombre de tours, soit par un registre (ROXL Dn,Dd permute Dd le nombre de fois indiqué par Dn), soit par une donnée immédiate (ROXL #,Dd avec une valeur de 0 a 7 pour la donnée). Si l'opérande est une case mémoire, un seul tour s'effectue et l'opérande ne peut être qu'un mot. : ROXL .
 
ROXRRotation droite avec le bit d'extension. Rotation étendue vers la droite de l'opérande. Le bit de poids faible est dupliqué dans les bits C et X du CCR qui, a son tour, est dupliqué dans le bit de poids fort de l'opérande traité. Si l'opérande est un registre on spécifie le nombre de tours, soit par un registre (ROXR Dn,Dd permute Dd le nombre de fois indiqué par Dn), soit par une donnée immédiate (ROXR #,Dd avec une valeur de 0 a 7 pour la donnée). Si l'opérande est une case mémoire, un seul tour s'effectue et l'opérande ne peut être qu'un mot. : ROXR .
 
RTERetour d'exception. Le registre SR est pris sur la pile, puis le PC est a son tour dépilé.
 
RTSRetour d'une sous routine. Le PC est récupéré sur la pile en le dépilant.
 
RTRRetour d'une sous routine et restauration de l'état initial du CCR (CCR lors de l'appel). Le PC et le CCR sont récupérés sur la pile en les dépilant.
