« site » : Représente un site web au sein du moteur.

Les arguments associés à l'élément « site »
nom Nom qui servira pour le référencement.
abrege Champ libre.
lang Selectionne le language
lang_select OUI / NON
title Champ libre.
home Champ de type URL.
directory Defini le répertoire ou se trouve les element relatifs au site
debug_info_level Niveau d'information dans le panneau d'administration.
stylesheet STATIQUE, DYNAMIQUE
etat OFFLINE, ONLINE, SUPPRIME, MAINTENANCE
filtre Champ libre. Permet de filter le résultat de la commande "show" sur le nom interne des éléments cible.

Exemple:

> add site name "MultiWeb_Manager" abrege "MWM" lang "FRA" lang_select "YES" title "M W M" home "www.rootwave.net" directory "00_mwm" debug_info_level "2" stylesheet "DYNAMIC" etat "ONLINE";
