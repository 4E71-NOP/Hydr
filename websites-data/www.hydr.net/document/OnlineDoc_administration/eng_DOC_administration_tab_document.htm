« document » : Designate a document.

Associated arguments of « document »
name Name used as internal reference.
state ONLINE / OFFLINE
type PHP, MIXED, WMCODE, NOCODE
filter Text field. Enable filtering on the internal name of the target elements for the "show" command line.
Used for inserting content (insert_content)
file Defines the file to be used.
to Define the target document.
Used for sharing (share_document).
with_site website
modification OUI/NON (YES/NO)
Used for updating (updfate)
checked OUI/NON (YES/NO)
examiner login
from_site website

Example:

> site_context "MultiWeb_Engine";
> add document name "doc_uni_correction_des_documents_p01.php" type "PHP";
> insert_content file "uni_correction_des_documents_p01.php" to "doc_uni_correction_des_documents_p01.php";
> update document name "doc_uni_correction_des_documents_p01.php" checked "YES" examiner "dieu" from_site "MultiWeb_Manager";
