« category » : Représente les éléments d'un menu.

Les arguments associés a l'élément « catégorie »
nom Nom qui servira pour le référencement.
parent Détermine la catégorie parente.
position Détermine la position de la catégorie dans la branche du menu.
type 4 types sont disponible : MENU_ADMIN_RACINE MENU_ADMIN ARTICLE_RACINE ARTICLE.
lang Détermine la langue de la catégorie. Ne sera donc affiché qu'en fonction de paramètre concordant.
etat ONLINE / OFFLINE
deadline Référence le deadline.
titre Champ libre. Apparaît dans le menu.
desc Description.
group Détermine quel group est autorisé a voir la catégorie et donc l'utiliser.
doc_premier Détermine l'article a utiliser si aucun n'est spécifié durant la navigation.
article Utiliser 0 pour en faire un répertoire; sinon donner le nom de l'article.
filtre Champ libre. Permet de filter le résultat de la commande "show" sur le nom interne des éléments cible.

Exemple:

> site_context "MultiWeb_Engine";
> add category name "fr_sommaire_admin" parent "0" position "01" type "MENU_ADMIN_RACINE" lang "FRA" state "ONLINE" deadline "initial_online" title "Sommaire" desc "Root menu" group "Anonyme" article "0" first_doc "YES";
