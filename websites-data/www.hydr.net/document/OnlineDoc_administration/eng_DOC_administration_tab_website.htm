« site » : Designate a website in the engine.

Associated arguments of « site »
name Name used as internal reference.
abrege Text field.
lang Defines language
lang_select OUI / NON (YES / NO)
title Text field.
home URL field.
directory Defines the working directory of the website
debug_info_level Information level in the admin panel.
stylesheet STATIQUE, DYNAMIQUE
state OFFLINE, ONLINE, SUPPRIME, MAINTENANCE
filter Text field. Enable filtering on the internal name of the target elements for the "show" command line.

Exemple:

> add site name "MultiWeb_Manager" abrege "MWM" lang "FRA" lang_select "YES" title "M W M" home "www.rootwave.net" directory "00_mwm" debug_info_level "2" stylesheet "DYNAMIC" etat "ONLINE";
