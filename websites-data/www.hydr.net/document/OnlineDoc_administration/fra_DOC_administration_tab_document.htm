« document » : Représente un document.

Les arguments associés à l'élément « document »
nom Nom qui servira pour le référencement.
etat ONLINE / OFFLINE
type PHP, MIXED, WMCODE, NOCODE
filtre Champ libre. Permet de filter le résultat de la commande "show" sur le nom interne des éléments cible.
Utilisé pour l'insertion de contenu (insert_content)
file nom_du_fichier
to Nom_du_document
Utilisé pour le partage (share_document).
with_site website
modification OUI/NON (YES/NO)
Utilisé pour la modification (updfate)
checked / verifie OUI/NON (YES/NO)
examiner / correcteur login
from_site website

Exemple:

> site_context "MultiWeb_Engine";
> add document name "doc_uni_correction_des_documents_p01.php" type "PHP";
> insert_content file "uni_correction_des_documents_p01.php" to "doc_uni_correction_des_documents_p01.php";
> update document name "doc_uni_correction_des_documents_p01.php" checked "YES" examiner "dieu" from_site "MultiWeb_Manager";
