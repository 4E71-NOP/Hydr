Basic commands
site_context This is one of the most important command. This command selects the website on which modification have to be applied.

Example:

> site_context "MultiWeb_Engine";
Add Adds an element. Needs the element name. Each element name has to be unique in its own category (user/deadline etc.).
Update Modifies an element. The element selection is bound to the condition that the element exists.
Delete Delete an element. The element selection is bound to the condition that the element exists.
Show Displays element details. The element selection is bound to the condition that the element exists.
Specialized commands
insert_content Replace the content of a document by the selectd file content
share_document Allows the sharing of a document. Only the website (context) owner can do it for his/its document and / or the server owner (role).
user Initiate an action on a user (ex: join_group).
Link Dedicated article command that allows to link an article with a document.
