« group » : Représente un group d'utilisateur.

Les arguments associés à l'élément « group »
nom Nom qui servira pour le référencement.
titre Champ libre.
tag STAFF, LECTEUR, ANONYME
file Champ de type ficher. Défini l'icone.
desc Description.
filtre Champ libre. Permet de filter le résultat de la commande "show" sur le nom interne des éléments cible.

Exemple:

> site_context "MultiWeb_Engine";
> Add group name "Server_owner" title "Server owner" tag STAFF file "graph/universal/icon_dev_001.jpg" desc "Omg owner lol";
