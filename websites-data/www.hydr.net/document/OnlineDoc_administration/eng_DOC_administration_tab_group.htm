« group » : Designate a group.

Associated arguments of « group »
name Name used as internal reference.
title Text field.
tag STAFF, LECTEUR, ANONYME
file File field. Défini l'icone.
desc Description.
filter Text field. Enable filtering on the internal name of the target elements for the "show" command line.

Example:

> site_context "MultiWeb_Engine";
> Add group name "Server_owner" title "Server owner" tag STAFF file "graph/universal/icone_developpeur_001.jpg" desc "Omg owner lol";
